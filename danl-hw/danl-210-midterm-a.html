Question 4

  • Write a Python code to identify which Name has powered homes most for each Year and Month.
    • In your Python code, please handle ties when filtering the observations with the multiple highest values of HomesPowered.
Code 
q4 = (
    trashwheel
    .groupby(['Name', 'Year', 'Month'])[['HomesPowered']]
    .sum()
    .reset_index()
    .groupby(['Year', 'Month'])
    .apply(lambda df: df.nlargest(1, 'HomesPowered', keep = "all"))
    .reset_index(drop = True)
)

Question 5

  • Each ton of trash equates to 500 kilowatts of electricity.

  • Write a Python code to determine the average monthly electricity supplied to homes by each Trash Wheel.

Code 
q5 = (
    trashwheel
    .groupby(['Name', 'Year', 'Month'])[['Weight', 'HomesPowered']]
    .sum()
    .reset_index()
    .assign(
        electricity = lambda x: x['Weight'] * 500
    )
)

q5['avg'] = q5['electricity'] / q5['HomesPowered']
q5 = q5.dropna(subset = ['avg']) # optional

show(q5)
